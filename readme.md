# C2-NET Correction

Activar e instalar el entorno de desarrollo
```
python3 -m venv venv
source venv/bin/activate
pip install requirements.txt
````

Generar las figuras
````
python figure_generation_script [FIGURA]
````
Si FIGURA no toma ningún valor, se generan todas las figuras. Si se da un valor 1-8 se generará la figura correspondiente.
